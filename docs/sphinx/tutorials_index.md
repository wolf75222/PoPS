# Tutoriels

Les parcours guidés (installation, diocotron, Poisson, API Python, AMR multi-niveau et
multi-patch, deux-fluides AP, physique magnétisée, backends) sont des cas d'usage applicatifs :
ils vivent désormais dans le dépôt
[`adc_cases`](https://github.com/wolf75222/adc_cases), qui importe le module `adc` fourni par
cette bibliothèque.

Pour démarrer côté bibliothèque, voir l'[installation](installation.md) et le
[quickstart](quickstart.md). Pour la référence de l'API Python, voir [api](api.md).
